## 🌐 What Is This Project?

This repository is my **Digital Literacy Portfolio** — built as part of the CSE0001 course at VIT Bhopal University.

The assignment put me in the shoes of a **Student Digital Ambassador**: someone whose job is to help fellow students navigate the internet safely, professionally, and confidently. Over five tasks — each tied to a module of the course — I explored everything from designing infographics and setting up professional profiles, to writing proper emails, practising coding, and understanding cybercrime.

No prior coding experience was needed. Every tool used here is free and browser-based. The GitHub repository you're reading right now is itself part of what I had to learn and submit.
